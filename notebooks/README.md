# Notebooks

This folder contains the Jupyter Notebook used for the climate analytics and regression modeling workflow.

## Main File

- `Model.ipynb`

## Purpose

The notebook demonstrates the analytical workflow used in this project, including:

- Data loading
- Data cleaning
- Feature preprocessing
- Exploratory analysis
- Regression model development
- Model comparison using Mean Absolute Error
- Interpretation of results

## Note

The datasets required to fully rerun the notebook are not included in this repository because dataset redistribution permissions have not been fully verified.

The notebook is included to show the methodology, code structure, modeling approach, and analytical reasoning used in the project.
