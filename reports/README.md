# Reports

This folder contains the final written report for the climate analytics project.

## Main File

- `Climate Analytics Project.pdf`

## Purpose

The report documents the complete project, including:

- Background and motivation
- Great Barrier Reef tourism context
- Climate change and coral bleaching indicators
- Data source descriptions
- Data preprocessing and modeling approach
- Regression model comparison
- Key findings
- Limitations
- Future work
- Bibliography

## Note

The report is provided for academic portfolio demonstration. It should be read together with the notebook and presentation to understand the full project workflow.
