# Data

The raw and processed datasets used in this project are not included in this repository because the dataset redistribution permissions have not been fully verified.

This project was completed as part of a college climate analytics project. The analysis used climate-stress, coral bleaching, tourism, GDP, and GSP indicators referenced in the final project report.

## Dataset Status

The following types of files are intentionally excluded from this repository:

- `.csv` files
- `.xlsx` files
- `.xls` files
- Raw data files
- Processed data files
- Large downloaded datasets

## Reason for Exclusion

The datasets may come from public academic sources, statistical sources, course-provided material, or modified team datasets. Since the redistribution rights are not fully confirmed, the datasets are not uploaded to keep the repository safe and professional.

## Where to Find More Information

Please refer to the final report in the `reports/` folder for:

- Data source descriptions
- Variable explanations
- Preprocessing steps
- Modeling methodology
- Model evaluation results
- Limitations and future work

## Future Update

If the datasets are later confirmed to be publicly reusable, they can be added to this folder with proper source attribution and license information.
