# Tableau

This folder contains the Tableau workbook used for data visualization.

## Main File

- `Coral Bleaching.twb`

## Purpose

The Tableau workbook was used to create visualizations related to climate indicators, coral bleaching context, and tourism analysis.

## Important Note

The Tableau workbook may require the original local datasets to open correctly. These datasets are not included in this repository because dataset redistribution permissions have not been fully verified.

If the workbook does not load correctly, reconnect it to the appropriate local or publicly available data source.
